Flashcard Creator
A simple web application for creating, managing, and generating printable flashcards. This tool allows users to input front and back text for flashcards, save them locally in the browser, and then download them as a PDF formatted for easy double-sided printing and cutting.

Features
Create Flashcards: Easily add new flashcards with separate front and back content.

Local Storage: All your flashcards are automatically saved in your browser's local storage, so they persist even if you close the tab.

Dark Mode: Toggle between a light and dark theme for comfortable use.

PDF Generation: Download all your created flashcards as a PDF document.

Print-Ready Format: PDFs are formatted on A4 portrait pages, with two A5 landscape flashcards per page (one above the other), optimized for double-sided printing and cutting.

Technologies Used
HTML5: For the basic structure of the web application.

CSS3: For styling and dark mode functionality.

JavaScript (ES6+): For all interactive functionalities, including card management and PDF generation.

jsPDF: A powerful client-side JavaScript library for generating PDFs. Used here for precise control over the PDF layout and text drawing.
